# `retry`

> Maximum retry attempts before the job is marked failed.

| | |
|---|---|
| **Kind** | Attribute |
| **Since** | `0.1.0` |
| **Default** | `0` |

## Syntax

```
retry <n>
```

## Arguments

| Name | Type | Required |
|------|------|----------|
| `n` | `int` | yes |

## Used in

- [`job`](../keywords/job.md)
