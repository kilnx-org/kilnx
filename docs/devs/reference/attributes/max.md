# `max`

> Maximum value (numeric) or length (string).

| | |
|---|---|
| **Kind** | Attribute |
| **Since** | `0.1.0` |

## Syntax

```
<field>: <type> max <n>
```

## Arguments

| Name | Type | Required |
|------|------|----------|
| `n` | `int` | yes |

## Description

Mirror of `min`. For string fields, this also influences column size where applicable.
