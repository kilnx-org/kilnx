# `custom`

> Load custom field definitions from an external manifest.

| | |
|---|---|
| **Kind** | Attribute |
| **Since** | `0.1.0` |

## Syntax

```
custom "<path>"
```

## Arguments

| Name | Type | Required |
|------|------|----------|
| `path` | `string` | yes |

## Description

Resolves at parse time from a path that may contain `{placeholder}` segments.

- [`model`](../keywords/model.md)
