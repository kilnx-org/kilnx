# `stream-swap`

> Hyperstream swap style (`append`, `inner`, `outer`, ...).

| | |
|---|---|
| **Kind** | Attribute |
| **Since** | `0.1.3` |
| **Default** | `append` |

## Syntax

```
stream-swap: <style>
```

## Arguments

| Name | Type | Required |
|------|------|----------|
| `style` | `enum` | yes |

## Description

Default `append`. Follows hyperstream's swap dispatcher whitelist.
